# Crypto-Lab-IRPET

This repository contains the code for the Cryptography Lab session held at IRPET in Florence on April 28th, 2026.

# Repository structure

The repository is organized as follows:

- `data`: This folder contains some example text files to test the cryptographic primitives.
- `slides`: This folder contains the slides used during the lab session.
- `results`: This folder will contain the files generated while running the Python notebooks.

The repository also includes a set of Python notebooks that implement various cryptographic primitives (i.e., encryption, decryption, hashing, digital signatures, Merkle trees) and demonstrate their usage.
